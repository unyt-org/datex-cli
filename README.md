# DATEX CLI

## Usage

### Running the REPL
```shell
datex
```

### Running a DATEX file
```shell
datex run path/to/file.dx
```

## Development
### Running the REPL
```shell
cargo run
```

### Running the Workbench
```shell
cargo run workbench
```

### Building for Release
```shell
cargo build --release
./target/release/datex_cli
```
